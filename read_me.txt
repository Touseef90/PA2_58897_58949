This directory have files for scanning and parsing Java programming language.

You need run command in terminal in the following order
1 - jflex example.jflex
2 - javac test.java
3 - java test input.java

It will show the result on terminal as well as write on output.txt file.

Note: Java version use by this project is.
java version "1.8.0_171"
Java(TM) SE Runtime Environment (build 1.8.0_171-b11)
